# Error Page
A creative and responsive error page inspired by a design I created in Figma.

## Demo
<img width="1907" height="916" alt="figmabearblue" src="https://github.com/user-attachments/assets/c9f22251-1422-44f3-be17-cb258cb9d61a" />

## Features
- Styled 404 error page with custom illustration
- Responsive layout for different screen sizes
- Elegant typography with the Inknut Antiqua font
- Use of multiple backgrounds for visual composition
- Simple structure and easy to adapt for other projects

## Tech Stack
**Frontend:** HTML5, CSS3
**GitHub Pages for deployment**

## Setup
1. Clone this repository
```
git clone https://github.com/rayssarrsilva/Error-page.git
```
2. Run locally by opening index.html in the browser

## What i learned
1. How to structure multiple backgrounds in CSS
2. Use of position and z-index to control layers
3. Trade-off between using <img> and background-image
4. Improvements in responsiveness and typography
